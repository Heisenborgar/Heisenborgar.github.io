Proof of Concept: Misuga Crew History List Excel Sorter/Filter

A Proof of Concept web app that reads exported Excel file from BTS Clearview's Crew History List Module.

How to use:

1. Login to ClearView.

2. Go to Reports > Basic Reports > Crew History List Module.

3. Select Fleet and Rank Filters.

4. Click "Export to Excel". An excel file should be downloaded afterwards.

5. Open the Misuga-PoC-ExcelSort website.

6. Upload exported excel file. Once uplaoded, the website will read the excel file.

7. Set the Start/End Date, Employment Status, and Vessel Type filters.

8. Click "Filter" button to show results.

9. Click "Clear Filters" button to reset filters and show the original, unfiltered excel file.

Note: Once "Filter" button is clicked, the website will only show results based on the specified filters. Furthermore, based on the selected Vessel type filter, that column will be highlighted with a distinct green color that gets lighter (until it's not visible) as the numbers go from highest (experienced) to lowest (inexperienced).
